# Mundo Tech - Primera Edición Digital

Este repositorio contiene la solución al taller práctico de HTML y CSS enfocado en la maquetación web estructurada, tipografías avanzadas, tablas estilizadas, listas y el comportamiento del modelo de cajas.

## Tecnologías Utilizadas
- **HTML5:** Marcado semántico (`<header>`, `<main>`, `<section>`, `<table>`, etc.).
- **CSS3:** Estilos de fuentes externas, manejo de desbordamientos (`overflow`), espaciados de texto y demostraciones de colapso de márgenes.
- **Google Fonts:** Uso de las familias tipográficas *Roboto* y *Playfair Display*.

## Conceptos Clave Demostrados
1. **Modelo de Cajas:** Aplicación analítica de `margin`, `padding` y `border` con bordes redondeados.
2. **Colapso de Márgenes:** Evidenciado en la sección final donde los márgenes contiguos verticales de 50px no se acumulan.
3. **White-space:** Comparativa de renderizado entre los valores `normal` y `pre`.
4. **Vertical Align:** Control de alineación de texto por celdas dentro de una tabla semántica.
